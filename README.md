# ULA 2 bits - Logisim

**Unidade Lógica Aritmética de 2 bits** com as 4 operações básicas (+, -, *, /) implementada apenas com portas lógicas.

- ✅ Multiplicação
- ✅ Divisão

3. Teste as operações usando as entradas **A** e **B** (2 bits cada)
